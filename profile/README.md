当你看到这个Org的时候，**恭喜你了解了SCU Maker（bushi）！**

SCU Maker的中文名叫开源硬件和智能基座协会，但是实际上我们很想改掉这个(╬▔皿▔)╯因为**我们真的不是只做硬件**

我们协会作为横跨**计软网电子机械自动化艺术历史等**专业的大社团，社团中有**各路大神**。

这里不仅有会写小程序的，会玩服务器的，会自动抢课的，会玩无人机的，会玩船的，会玩机器人的，会3D打印的，会翻墙的，会玩AI的...还有会画画的，会剪辑的，会玩VR游戏的。
