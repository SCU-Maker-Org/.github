## ᓚᘏᗢ
​        四川大学开源硬件协会成立于2015年9月，是一个科技类学术社团，始终坚守“传递科技薪火，发扬开源精神”的初心，致力于将计算机软硬件知识转化为实际应用，帮助同学们将创意变为现实。

​       在活动方面，协会开发了“帷胜-端边协同的无人载具控制系统”，在第七届“互联网+”大赛中获得全国银奖，并与企业合作服务成渝双城经济圈建设。协会还在核电站水下辐射环境相关项目中开展研究，设计的算法满足目标精确测量需求。此外，协会将可见光通信技术应用于川大校史馆智慧展览系统，为学校红色资源赋能，并开设了全国首家学生自主运营的无人超市“极市”，获教育部官网报道。协会还积极带领同学们参加各类高水平比赛，获得了华为ICT创新大赛全球一等奖、2024年全国大学生物联网设计竞赛—华为杯等奖项。

​        在荣誉方面，协会获得国家级奖项50余项，省级奖项40余项，并被授予成都市百佳社团、四川大学“十佳学生集体”称号。协会成员还通过公益活动回馈母校和社会，如开展“从零开始的创客之旅”系列讲座，覆盖十余学院、两千余学生，带动更多同学参与创新创业。

​       协会以“开源精神”为核心，倡导奉献、开放、共享和创新，致力于推动科技知识的普及和开源事业的发展。未来，协会将继续立足川大校园，为更多学生提供科技创新的平台，助力川大学子在科技领域不断前行。
